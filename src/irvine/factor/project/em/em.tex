\section{Conjectures}

\begin{enumerate}

\item Each Euclid-Mullin sequence contains every prime.

\item All Euclid-Mullin sequences are ultimately the same.

\item For each prime $p$ and fixed constant $m$, there exists an Euclid-Mullin sequence in which $p^{m}$ divides a term.
